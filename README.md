# 🧩 Mini Apps – Multi Tool Web Application

<img src="thumbnail.png" alt="Mini Apps Preview" width="700">

**Mini Apps** is a modern multi-utility web application that combines four interactive tools in one website:

✅ To-Do List  
✅ Quiz App  
✅ Calculator  
✅ Weather App  

The project is built using **HTML, CSS, and JavaScript** with a clean dark UI, smooth animations, and responsive design.

---

## ✨ Features

- ☑ Task management with To-Do List
- ❓ Interactive JavaScript Quiz
- ⌗ Functional Calculator
- ⛅ Real-time Weather Search (API based)
- 🎨 Modern dark UI design
- 📱 Fully responsive layout
- ⚡ Smooth tab navigation
- 🔥 Single-page multi app system

---

## 🧰 Apps Included

### ☑ To-Do App
- Add tasks
- Mark completed
- Delete tasks
- Progress tracking

### ❓ Quiz App
- Multiple-choice questions
- Score system
- Progress indicator
- Restart option

### 🧮 Calculator
- Basic arithmetic operations
- Percentage & sign toggle
- Clean UI calculator

### ⛅ Weather App
- Search weather by city
- Temperature & humidity
- Wind speed & visibility
- OpenWeatherMap API integration

## ⚙️ How to Run Locally

1. Clone the repository

git clone https://github.com/fsafiya187/mini-apps.git


2. Open project folder

cd mini-apps


3. Run the project

Open `index.html` in your browser.

---

## 🌦 Weather API Setup

This project uses **OpenWeatherMap API**.

Steps:
1. Go to https://openweathermap.org
2. Create a free account
3. Generate API Key
4. Paste the API key inside the Weather section input
